Place your logo image here so the site can use it as the 3rdeyeslime logo.

Save the attached image (from the editor) as:

  public/images/logo.png

Optional:
- If you'd like the same artwork as a hero background, save a copy as:
  public/images/welcome-bg.png

Notes:
- The app will try to load `/images/logo.png`. If the file is missing or fails to load, the page will automatically show a small inline SVG fallback logo instead.
- Recommended formats: PNG (lossless) or WebP for smaller size. For logos, SVG is preferred when available.
